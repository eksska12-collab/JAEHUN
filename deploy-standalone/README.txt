마케팅 10팀 출퇴근 현황 - 배포용 폴더

===========================================
이 폴더를 Vercel이나 Netlify에 드래그 앤 드롭하세요!
===========================================

배포 방법:

1. Vercel 사용:
   - https://vercel.com/new 접속
   - 이 폴더(deploy-standalone)를 드래그 앤 드롭
   - "Deploy" 클릭

2. Netlify 사용:
   - https://app.netlify.com/drop 접속
   - 이 폴더(deploy-standalone)를 드래그 앤 드롭
   - 즉시 배포 완료!

3. 로컬에서 테스트:
   - index.html 파일을 브라우저에서 열기

===========================================

모든 기능이 작동합니다:
✓ 날씨 정보
✓ 상태 관리
✓ localStorage 저장
✓ 모바일 최적화

배포 후 URL을 팀원들과 공유하세요!
